# Portfolio Website

A modern, responsive personal portfolio website for Gerand C. Parawan, built with HTML5, CSS3, and JavaScript.

## Features

- Responsive layout with mobile-friendly navigation
- Light and dark theme toggle
- Smooth scrolling and reveal animations
- Project showcase and grouped skills section
- Accessible semantic structure with modern CSS

## Files

- `index.html` — main page markup
- `styles.css` — theme and layout styling
- `script.js` — theme toggle, mobile menu, and section reveal logic
- `assets/profile-placeholder.svg` — placeholder profile image

## Usage

1. Open `index.html` in your browser.
2. Replace the placeholder image with your own profile photo.
3. Update project links, content, and contact details as needed.
